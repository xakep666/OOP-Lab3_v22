std::vector< std::string > Application::abonentInfo ( ulong  abonentaddr) const

Cбор информации о количестве переданных данных и длительности связи для абонента

Аргументы
abonentaddrIP адрес абонента
Возвращает
vector из string - текстовое представление собранной информации
Исключения
invalid_argumentесли адрес абонента неверен или не найден

Метод проходит по всем серверам и их таблицам связи, используя итераторы, собирая информацию во "временную" коллекцию следующего вида

std::vector<std::pair<ulong,ulong>> counter;
counter.reserve(ServicesNum);

Первый элемент пары - суммарный траффик для сервиса (передано+получено), Второй - Общая длительность связи для сервиса (не для всех сервисов) Вектор result резервируется для 3-х сервисов

void Application::addService ( ulong  serveraddr,
ulong  abonentaddr,
ServiceDescriptor *  sdesc 
)

Добавление записи об оказанной услуге в "таблицу связи" сервера

Аргументы
serveraddrIP адрес сервера
abonentaddrIP адрес клиента
sdescУказатель на описатель сервиса (полиморфный ServiceDescriptor)
Исключения
invalid_argumentЕсли:
  • Неверный IP сервера
  • Неверный IP клиента
  • В качестве указателя на ServiceDescriptor передан нулевой указатель

Метод ищет сервер в таблице серверов и добавляет в его "таблицу связи" информацию об оказанной услуге. Также метод добавляет IP адрес клиента в таблицу занятых адресов, если такого адреса там еще нет

template<class T>
class NetworkService::MyConstIterator< T >

Шаблонный класс итератора

Первый параметр шаблона - класс, к которому прикрепляется итератор Для корректной работы требуется следующее:

  • Переопределенный тип indexT (тип, возвращаемый оператором [])
  • Перегруженный оператор [] Отличается от обычного итератора тем, что не позволяет изменять значение элемента, на который он указывает

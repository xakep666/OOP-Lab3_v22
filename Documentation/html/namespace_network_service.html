bool NetworkService::isValidIP ( ulong  ip)

Проверка, является ли IP адрес верным

Аргументы
ipIP адрес в ulong-представлении
Возвращает
true если IP верный, false в противном случае

IP адрес считается верным, если он не начинается и не заканчивается на 0 или 255. Проверка происходит с использованием битовых сдвигов, побитовых "и" и сравнений. Пример реализации проверки

if ( (ip&0xFF)==0 || (ip&0xFF)==0xFF)
return false;
if (((ip>>(IPSize-8))&0xFF)==0 || ((ip>>(IPSize-8))&0xFF)==0xFF)
return false;
return true;
